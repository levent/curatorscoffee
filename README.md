# Curators Coffee

## About this site

This site is powered by [Jekyll](http://gruntjs.com/), [Grunt](http://gruntjs.com/) and [coffee](http://www.curatorscoffee.com).

## Prerequisites

* Ruby
* Node.js

## Getting Started

1. bundle install
2. npm install
3. bundle exec grunt serve
